作品说明：
ESA 短链接服务 - 基于阿里云ESA边缘计算的现代化短链接生成服务

【项目简介】
本项目是一个完整的短链接服务应用，充分利用阿里云ESA边缘计算能力，提供快速、安全、可靠的URL缩短功能。

【核心功能】
1. 快速生成短链接 - 输入长URL，一键生成短链接
2. 自定义短码 - 支持自定义短链接别名，便于记忆和分享
3. 二维码生成 - 自动生成二维码
4. 点击统计 - 自动记录短链接点击次数，实时查询
5. 过期时间 - 支持设置链接有效期（1小时/1天/7天/30天/永久）
6. 活码功能 - 使用管理密钥随时修改短链接目标URL，无需重新生成
7. 最近创建 - 本地存储最近创建的5条短链接记录

【技术架构】
1. ESA Pages 静态托管 - 使用 esa.jsonc 配置自动构建和部署，支持 SPA 单页应用模式
2. ESA 边缘函数 (Edge Functions) - 实现短链接生成、重定向等核心功能，边缘计算，低延迟响应（<50ms）
3. ESA Edge KV 存储 - 全球分布式键值存储，数据 20 秒内全球同步，持久化短链接数据和点击统计
4. Vue 3 + Vite 技术栈 - 使用 Vue 3 Composition API，Vue Router 实现前端路由，Vite 构建工具

【技术亮点】
1. 完整的边缘计算架构 - 完整使用ESA生态（Pages + Functions + KV），真正的 Serverless 架构，无需服务器
2. 全球分布式部署 - 数据存储在全球边缘节点，20 秒内全球同步，用户访问就近节点，低延迟（<50ms）
3. 现代化开发体验 - Vue 3 Composition API，Vite 快速构建，热模块替换（HMR）
4. 用户体验优化 - Toast 通知替代原生弹窗，现代 Clipboard API 支持，优雅的加载状态，流畅的动画效果

【界面设计】
- 现代化UI设计：清爽简洁的界面设计，蓝色系配色
- 流畅动画：卡片悬停动画、按钮交互动画，提升用户体验
- 响应式设计：完美适配桌面、平板、手机等各种设备
- 两列布局：主操作区域采用两列布局，提升操作效率
- 功能说明：顶部功能特色卡片，清晰展示所有功能

【应用价值】
本项目是一个高度实用的短链接服务，部署后立即可用，无需任何配置。支持自定义短码、QR码生成、一键复制、全球加速等功能，基于ESA边缘节点，全球访问超快响应。适合个人和团队使用，可用于营销推广、社交分享、数据统计等多种场景。
